# rayTRACING
Gamagora 2019 - Ray Tracing

*Ray Tracing C++*

- Affichage de Sphères
- Eclairage direct (Point Lights / Lampes surfaciques)
- Eclairage indirect
- Cornell Box
- Miroirs
- Vitres
- BVH
