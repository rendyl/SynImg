# rayTRACING
Gamagora 2019 - Ray Tracing

*Ray Tracing C++ from scratch with FreeImage library*

## Actual Features :
- Display Spheres
- Normal Lighting (Point Light / Surface Light)
- Indirect Lighting
- Cornell Box
- Mirror
- Glass
- BVH

## In Progress :
- Mesh Display
- BVH for Meshes
- Sampling Pixels
- Test BVH
